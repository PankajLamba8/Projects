#  Netflix Business Problem Analysis using SQL

##  Overview

This project focuses on solving real-world business problems using SQL analysis on Netflix content data.

The project includes database schema creation, business queries, and analytical problem-solving to generate insights from Netflix movies and TV shows data.

The analysis helps uncover:
- Content trends
- Genre popularity
- Country-wise distribution
- Ratings analysis
- Release patterns
- Content strategy insights

---

##  Project Objectives

- Analyze Netflix content database using SQL
- Solve business-oriented analytical questions
- Extract meaningful insights from streaming data
- Practice advanced SQL querying techniques
- Understand entertainment industry trends through data

##  Business Problems Solved

The project includes solutions for multiple analytical business questions such as:

- Most common content ratings
- Top genres available on Netflix
- Country-wise content production
- Movie vs TV show analysis

- Longest movies analysis
- Director and actor-based insights

##  Key Insights

- Netflix content production increased significantly after 2015
- Movies dominate the platform compared to TV shows
- Certain countries contribute most of the available content
- Drama and comedy genres appear most frequently
- Content ratings vary across regions and categories
